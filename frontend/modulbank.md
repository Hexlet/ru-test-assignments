Вам предстоит создать мини веб приложение, используя паттерн “Конечные автоматы” для управления состоянием интерфейса.

Посмотрите обучающее видео, в котором с помощью библиотеки XState создается управление состояниями видеоплеера.

https://www.youtube.com/watch?v=3wZBSeLxVEw

Реализуйте такой же функционал, используя [React](https://stately.ai/docs/xstate-react)  (
) или [Vue](https://stately.ai/docs/xstate-vue)

В видео с примером используется старая версия XState 4. Вы можете использовать версию 4 или 5.

В качестве готового компонента для окна плеера можно использовать [компонент Modal](https://ant.design/components/modal) из библиотеки Ant

В качестве видеоплеера можно использовать [react player](https://github.com/CookPete/react-player) или любой другой
[Ссылка на видеопоток](https://cdn.flowplayer.com/d9cd469f-14fc-4b7b-a7f6-ccbfa755dcb8/hls/383f752a-cbd1-4691-a73f-a4e583391b3d/playlist.m3u8) (можно использовать любой другой)

Добавьте возможность переключения плеера на уменьшенную версию окна. Логика переключения между состояниями должна быть реализована в парадигме State Machines, с добавлением новых состояний в XState.

Образец того что должно получиться находится по адресу https://test-modulbank.web.app/

Если у вас возникло желание внести улучшения в интерфейс или добавить еще какой-нибудь функционал - отлично, можете улучшить результат по своему предпочтению, это будет в плюс

Разместите ваш вариант решения на GitHub, желательно с несколькими коммитами, с помощью который можно будет отследить шаги при реализации.
