Задание
Разработать простой web чат на html/php с использованием PSD макетов и разместить на любом хостинге.

* Сверстать чат на основе макета <https://www.invisionapp.com/chat#>! (он же доступен по адресу)
* Перед входом в чат пользователю предлагается ввести свое имя, которое в дальнейшем используется для идентификации его сообщений
* Реализовать сохранение сообщений в памяти (на локальной машине, в localstorage или еще куда-то, без бекенда)
* Разрешается использовать любые javascript библиотеки и фреймворки
* БОНУСНОЕ ЗАДАНИЕ: разработать серверную часть для сохранения сообщений и, собственно реализации многопользовательского чата (например, на php или на любом другом языке) БД использовать по желанию, сохранение сообщений в файле для решения задачи подойдет.
* Выложить результат куда-то по общедоступной ссылке, чтобы мы могли посмотреть результат без разворачивания БД и устанавливки пакетов.
* Засечь время на выполнение задания и прислать результат через форму <https://airtable.com/shr10tjcREHUsNZGD>
