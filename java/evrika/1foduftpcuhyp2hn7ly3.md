Тестовое задание на должность Full Stack разработчика в АО «Эврика»

Необходимо разработать single page application, которое будет отображать список зарегистрированных пользователей в системе с возможностью создания новых пользователей и редактирования существующих. Вход в систему для просмотра списка пользователей должен осуществляться любым пользователем списка. Без входа в систему работа с приложением должна быть невозможна.

1. В качестве front-end использовать фреймворк ExtJs.
2. В качестве серверной части необходимо использовать языки/технологии/фреймворки:

* Java версии 8 или выше
* Spring
* Maven
* Hibernate

3. В качестве БД можно использовать любую БД, с которой умеет работать Hibernate. В базе данных будет достаточно хранить только пользователей системы.
4. В качестве сервера для запуска приложения использовать Apache Tomcat
5. Приложение должно легко собираться.
6. Если необходимо – приложить инструкцию для сборки и запуска приложения.
7. В качестве решения необходимо прислать

* ссылку на GitHub c исходниками
* результат сборки Maven (war-файл или Spring Boot jar), чтобы можно было запустить и проверить работу приложения без сборки проекта.
В качестве среды разработки необходимо использовать Intellij Idea.
